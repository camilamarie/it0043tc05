P O R T F O L I O

GUARDACODE - A Cybersecurity Application

Guardacode is a web application for learning the fundamentals of cybersecurity. It is a platform for students to learn and grow as a cybersecurity professional in a fun and interactive way with the use of different methods and gamified quizzes.

CyberZ - A Cyberthreat real-time map

CyberZ will find out if you are under cyber-attack. It allows users to compare different threats and their distribution around the world at any given time. It has Intrusion Detection Scans (IDS) and has a real time interactive map.

SECURIT - A Cybersecurity Mobile Application for Capture the Flag Challenges

Securit is a gamified mobile learning platform. Where the Testers guide participants through practical exercises, demonstrations, and simulations that mirror real-world cybersecurity scenarios.
